モーフタグ

モーフタグで、違う位置のメッシュを記録する事が出来ます。例えば、微笑んでる、悲しんでいる、笑っている、起こっている顔を記録しておく事が出来ます。モーフターゲットを記録した後で、これらの異なる表情（モーフ）の中間にモーフさせる事も出来ます。
またモーフターゲットで、顔を喋らせたり、筋肉を収縮をアニメーションさせる事もできます。

モーフターゲットを記録するためには、まずポリゴンオブジェクトにモーフタグを追加しなければなりません。そして、すべての表情（モーフ）を記録するまで、次のこのような手順に従って下さい。

  1. モーフプロパティの左下の "+" ボタンをクリックして、"Base" モーフターゲットを追加します。
  2. 新しい表情（モーフ）になるようにポリゴンメッシュを修正します。修正を、メッシュの頂点（ポイント）の移動のみにする事は重要です。ポリゴンの追加・分離・削除でメッシュトポロジーを変更しないでください。
  3. "+" ボタンで新しい表情（モーフ）を記録し、他のモーフと重ならない名称をつけます。
  4. すべてのモーフターゲットを記録するまで、手順 2 を続けます。
  5. 最後に "Base" モーフを選択し、プロパティの一覧表のコンテキストメニュー（右クリックメニュー/Alt+クリックメニュー）から "Restore" メニューコマンドを選択します。

これで、中間のモーフさせるためにスライダーを使用する事が可能になります。

プロパティ

  • モーフターゲット 記録されたモーフターゲットの一覧表。それぞれのモーフターゲットのウェイトを調整するには右のスライダーを使用します。
