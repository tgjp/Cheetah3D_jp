UV Editor

The UV editor displays the uv coordinate of polygon objects and the current editable texture. To edit the UV coords of a raw polygon object you just have to select it in the object browser or in the 3D view. But please be aware that the UV coords of parametric or creator objects can't be edited while it is possible to view them.

The position and scale of the view can be changed conveniently by the small icon bar above the UV editor. The functions of these icons from left to right are:

  • Focus on selected UV coords
  • Move view to the left, right, up and down
  • Zoom view in and out

The view position can also be changed by pressing a key combination + dragging the mouse in the 3D view:

  • If you have a one button mouse:
    • alt + mouse drag: move view
    • alt + command + mouse drag: zoom view
  • If you have a three button mouse with scroll wheel:
    • middle mouse button drag: move view
    • scroll wheel: zoom view

When you click inside the UV Editor, what happens depends largely on which tool is currently active.
