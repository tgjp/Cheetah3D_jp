UVマップをPDFで保存 ツール

Cheetah3Dの ペイントツールはCheetah3Dの中でほとんどのペインティング作業ができるのですが、全ての２次元ペインティング作業の置き換えができるわけではありません。そのため、時々他のアプリケーションを使ってペインティング作業を行ったり微調整作業を行う事があるかもしれません。
UVメッシュの下絵が無い状態でペインティング作業を行うのは難しいので、このツールを使いUV座標系をUVメッシュの下絵としてPDF形式で書き出すことが出来ます。その後に他のペインティングアプリケーションに書き出したPDFファイルを取り込み下絵として使用できます。


Cheetah3DでのUVメッシュ


Photoshopに取り込んだUVメッシュ

モード
 このツールは ポリゴンオブジェクト にのみ使用できます。


  • "UVマップをPDFで保存 " ツールを使うために ポリゴンオブジェクト を選択してください。そしてUV エディタを使って選択した全ての物がUV座標系の [0,1]内にあることを確認してください。 間違った範囲にあるとUV座標系のスケールを間違った状態で変換作業を行ってしまいます。最後に"UVマップをPDFで保存" ツールを選択してPDFファイル名を入力してください。
