UV マッパーツール

UV マッパーツールは基本投影形式の球体マッピングや立方体マッピングを適用したポリゴンオブジェクトにUV値を付与することができます。例えば、人の顔のような複雑なオブジェクトなどのUV 展開を行う場合はUV 展開ツールを使う事ができます.

Modes
 UV マッピングツールは、ポリゴンオブジェクトにのみ使うことができます。また、UV マッピングツールはポリゴンモード選択時のみ使用ができます。


  • UV マッピングツールを使用するために、いくつかのポリゴンまたはポリゴンオブジェクトを選択してください。それからメニューコマンド "ツールUVツールUV マッパー"を選んでください。その後、UV マッピングツールで必要な全てのパラメータをツールプロパティーエディタで設定してください。 そして、あなたが希望するUV座標値になるようにトランスフォームギズモを使いUV マッピングの格子を配置してください。編集の状態は UV エディタ または3D ビュー画面で確認することが出来ます。しかし、3D ビュー画面 でUV マッピング状態を確認するためには、あらかじめポリゴンオブジェクトにテクスチャーマップを割り当てておく必要があります。"UV 座標の書き込み" ボタンをクリックするとポリゴンオブジェクトに計算されたUV 座標値が割り当てられます。

キー

    • ESC: ツールの初期化。

プロパティー

  • マッピングタイプ:
    • 立方体マッピング: 立方体に投影するマッピング.
    • 球体マッピング: 球体に投影するマッピング.
    • 平面マッピング: 平面に投影するマッピング.
    • 円筒マッピング: 円筒に投影するマッピング.
    • 正面マッピング: 画面上のシーンに対して正面にある平面に投影するマッピング
  • 選択範囲のみ:: UV 座標値が全てのポリゴン、または選択した範囲のポリゴンにのみに割り当てるか指定します。
  • オフセット: 新しいUV 座標値をオフセットします。
  • 拡大縮小: 新たに計算したUV座標値を追加します。
  • 中心に移動: UV マッピング座標を選択しているポリゴンの中心に設定します。
  • UV マップの書き込み: このボタンを押すとマッピング形式に従ったUV座標値をポリゴンに設定します。

例
 選択したポリゴンにUV 座標値を設定します。

  1. 編集したいポリゴンオブジェクトを選択します。
  2. ポリゴンモードに変更します。
  3. UV 座標値を設定したいポリゴンを選択します。
  4. UV マッパーツールを選択します。
  5. UV マッパーツールのプロパティで選択範囲のみ をチェックします。
  6. マッピングタイプを選択します。
  7. トランスフォームギズモでUVマッピングの格子を移動します。
  8. 完了したらUV マップの書き込み ボタンを押します。これでUV座標値がポリゴンオブジェクトに設定されました。
  9. 全てのポリゴンの設定が終わるまで3項から繰り返します。
