UV 展開 ツール

UV 展開ツールは人間のように複雑な形状にUV座標値を設定する際には強力なツールとなります。 UV 展開アルゴリズムはテクスチャーの伸縮が最小になるように展開します。


Unwrapped polygon object.

モード
 UV 展開ツールは編集中のポリゴンにのみ使用できます。


  • まず、最初に行わなければならないのは UV 展開 ツールによって 切れ目 を設定する事です。（上図の緑色のエッジです。） 切れ目 は UV 展開 を行う際にめくりあげられるエッジです。 切れ目 を設定した後に "ツール->UV ツール->UV 展開" ツールを選択します。UV エディタでは メッシュのUV展開後の形状の確認が出来ます。UV ピンの設定/解除を併用することで 展開したメッシュの形状を変化させることが出来ます。
    UV 展開 チュートリアルを順番に実践して現実際の問題について体験してください。
