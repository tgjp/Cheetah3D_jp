回転 (法線を軸に) ツール

回転 (法線を軸に)ツールは、ポリゴンの法線ベクトルを軸に回転するために使用する事ができます。複数のポリゴンを選択している場合には、予想している結果にならない場合があるので注意して下さい。


編集モード
回転 (法線を軸に)ツールは、ローポリゴンオブジェクトにのみ使用する事ができます。


  • 回転 (法線を軸に)を実行するには、まず回転させたいポリゴンを選択ツールツールで選択します。次にメニューコマンド "ツールポリゴン回転 (法線を軸に)" からを選ぶかホットキーで回転 (法線を軸に)を選択します。３Dビュー上で、ドラッグ＆ドロップすると選択中のポリゴンがその法線ベクトルを軸に回転させる事ができます。マウスボタンをはなして、編集を決定します。
