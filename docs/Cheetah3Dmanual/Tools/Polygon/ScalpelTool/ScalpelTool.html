ナイフ ツール

ナイフツールは、ポリゴンメッシュを編集するのにとても便利なツールです。あるエッジ／ポイントから、他のエッジ／ポイントまで自由にカットするために使う事ができます。

編集モード
 ナイフツールは、ローポリゴンオブジェクトにのみ使用する事ができます。ポイントモード、エッジモード、ポリゴンモードで使用する事ができます。


  • すべてのカットには、始点と終点が必要です。始点を選択するには、エッジかポイントをクリックします。ポイントを選択すると、そのポイントはブルーで表示されます。エッジの任意のポイントを選択すると、グリーンで表示されます。次にポリゴン上を好きなだけポイントを追加しながらカットする事ができます。この作業を終了するには、同じポリゴン上のポイントかエッジを終点として選択します。これでポリゴンは始点から終点までのカットラインで２つに分離されます（上のイメージを見て下さい）。カットが終わった時点で終点は、自動的に次のカットの始点として選択されます。

補助キー

    • ESC: ツールをリセットするには、ESC キーを使います。
