Normal Scale Tool

The normal scale tool scales a polygon selection without destroying the shape of the geometry. The name Normal Scale doesn't imply that the normals are scaled, but that the tool tries to retain the normals of the geometry during the scale operation.


Modes
The normal scale tool can only be applied to raw polygon objects.


  • To perform a normal scale, select the polygons you want to scale with the select tool. Now choose the normal scale tool with the menu command "ToolsPolygonNormal Scale" or a hotkey. You can now drag and drop your selected polygons perpendicular to their normals within the 3D view. Release the mouse button to finish the operation.
