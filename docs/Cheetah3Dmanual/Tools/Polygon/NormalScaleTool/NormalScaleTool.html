拡大縮小 (法線に垂直な面で)ツール

拡大縮小 (法線に垂直な面で)ツールは選択したポリゴンの形状を破壊することなくスケールの変更が出来ます。名称の 拡大縮小 (法線に垂直な面で)は法線方向のスケール変更を含みません。しかし、このツールはスケール変更時に形状の法線方向を維持します。


モード
 拡大縮小 (法線に垂直な面で)ツールは ポリゴンオブジェクトにのみ使用できます。


  • 拡大縮小 (法線に垂直な面で)を使用するためには選択ツールを使いポリゴンを選択してください。そして、メニューコマンドから "ツールポリゴン拡大縮小 (法線に垂直な面で)"を選択するかホットキーを押下してください。これで、3D ビュー内で選択したポリゴンを法線に対し直角方向にドラッグドロップできます。マウスボタンから手を離すとこの操作は完了します。

キー

    • 無し

プロパティー
