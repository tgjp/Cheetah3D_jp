ポイントを接合ツール

ポイントを接合ツールは二つのポイントを接合する時や、エッジを一点に収束するときに使用できます。


モード
 ポイントを接合ツールは編集中のポリゴンオブジェクトのみに適用できます。ポイントモード、エッジモード、ポリゴンモードで利用可能です。


  • ポイントを接合ツールを使用するには、まず編集中のポリゴンオブジェクトの一番目のポイントをクリックしてください。すると、選択されたポイントは青く表示されます(一番目の画像を参照)。次に、接合したい二番目のポイントをクリックしてください。二つのポイントが接合されました(二番目の画像を参照)。

    もし、n 個のエッジを持つポリゴン上の二つのポイントに適用した場合、ポリゴンは n-1 個のエッジを持つポリゴンになります。三角形上の二つのポイントに適用した場合は、その三角形は削除されます。

キー

    • ESC: ESC キーを使ってこのツールをリセットします。

プロパティ

  • 接合位置: "中間点" を選択した場合、接合された新しいポイントは元の二つのポイントの中間の位置になります。"最初に指定したポイント" を選択した場合、最初にクリックしたポイントの位置が接合された新たなポイントの位置になります。
