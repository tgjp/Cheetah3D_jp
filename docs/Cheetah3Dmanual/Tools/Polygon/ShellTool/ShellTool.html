厚みをつけるツール

あなたが家のモデル作成のために壁を作る場合、あるいはあ平らな表面からオブジェクトを作成する場合は、厚みをつけるツールは選択のツールです。下図は、単純なポリゴンを厚みをつけるツールを使用して、どのようにリアルな壁を作成したかの説明図です。

モード
 厚みをつけるツールは編集中のポリゴンオブジェクトだけに適用することができます。また、ポリゴンモードでのみ使用できます。


  • 厚みをつけるツールを使用するために、編集中のポリゴン から一部を選んでください。その後、 厚みをつけるツールを選択してマウスを3D ビュー画面内で左右にドラッグし、厚みを付ける方向を選択してください。作業が完了したらマウスボタンを押すのをやめてください。

キー

    • 無し

プロパティ

  • 内側に: この設定値は厚みを付ける方向を内側にするか外側にするかの方向を設定します。
  • 角度: 隣り合うポリゴンの厚み方向を設定します。大きな値を設定するとより多くのポリゴンが集まります。
  • クォンタイズ: 大きな値を設定すると厚みをつける時の吸着間隔が大きくなります。この値は厚みを付ける際の吸着間隔を設定します。
