Merge Tool

The purpose of the merge tool is to combine two or more objects to one. You can use it to either combine polygon or spline objects. However, you can't mix up the two different types.

Modes
This tool only works on polygon or spline objects.


  • The merge tool is a headless tool which means that it doesn't show up in the tool view. To use it select at least two polygon or spline objects and call the tool from the menu "ToolsMerge". Cheetah3D then creates a new object by merging the geometry of the selected objects.
