Cheetah3D は、Mac OS X 用の簡単に使いこなせる、スリムで速く、洗練された 3D モデリング、レンダリング、アニメーションツールです。高度なサブディビジョンモデリングを含む強力なポリゴン編集ツールから、ブーリアン操作、ベジェスプラインまで多くのツールを提供しています。

Cheetah3D のそれぞれの機能をより理解するためには、下のトピックをクリックします。
