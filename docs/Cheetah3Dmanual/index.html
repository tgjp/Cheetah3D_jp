Cheetah3D is a lean, fast and elegant 3D modelling, rendering and animation tool for Mac OS X with an easy learning curve. It offers many tools from powerful polygon editing over advanced subdivision modelling to boolean operations and Bezier splines.

To find out more about each of the individual features of Cheetah3D, click one of the topics below.
