F-カーブエディタ

まだ、タイムラインとキーエディタについてお読みでない場合は、これらの項目を先にお読みください。

F-カーブエディタは、アニメーションを微調整する際の大変強力なツールです。あなたが既にアニメーションを記録している場合、選択されたオブジェクト、タグ、またはマテリアルのF-カーブエディタが表示されます。もしまだ記録していない場合、このビューは空白でしょう。

F-カーブエディタのコントロール:
 上のスクリーンショットに示されている、F-カーブエディタの重要な４つの部分から説明しましょう。

  1. 記録されているキーのパラメータ。選択されたパラメータのF-カーブのみをF-カーブビューに表示します。パラメータを選択するには、これをクリックします。
  2. 選択されたキー（訳注：実際に選択されているキーは緑のライン上の赤丸）
  3. 左側の接線ハンドル。スプライン補間を使用している場合のみ接線ハンドルが表示されます。
  4. 右側の接線ハンドル。スプライン補間を使用している場合のみ接線ハンドルが表示されます。

補間タイプ:
 Cheetah3D は、今のところ４つの補間タイプを提供しています。補間タイプは、どのようにキーの間のプロパティーの値を計算するかを定義します。

 リニア補間: これは最も基本的な補間のタイプです。パラメータは直線的に（同じスピードで）変化します。これは Cheetah3D のデフォルトの補間タイプです。（訳注：4.3 では、デフォルトはソフト補間です。）
ソフト補間: ソフト補間タイプは、それぞれのキーを通る滑らかな曲線を作ります。
スプライン補間: スプライン補間は、最も強力な補間タイプです。２つのキーを通る任意のベジェ曲線を定義する事ができます。もしスプライン補間を選んだ場合、F-カーブエディタにベジェ曲線の接線ハンドルが表示されます。この接線ハンドルを操作するには、マウスでドラッグします。
フラット補間: フラット補間タイプは、パラメータのスピードが曲線の始まりか終わりで 0 になるので、ゆるかやに加速、またはゆるやかに減速させたい場合に便利です。

補間タイプの設定:
 キーに補間タイプを設定するには、まずマウスで設定したいキーを選択します。選択されたキーは、赤でハイライトされます。メニュー（または右クリックメニュー）から、両側の接線タイプを設定する場合は、"アニメーション -> 接線 -> タイプ"、左側の接線タイプを設定する場合は、"アニメーション -> 接線 (入) -> タイプ", 右側の接線タイプを設定する場合は、"アニメーション -> 接線 (出) -> タイプ" を呼び出します。任意の異なる補間タイプを組み合わせる事も可能です。
