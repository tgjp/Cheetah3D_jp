Sweep Object

The sweep object allows you to make any kind of tube or hose. It's also great for creating some advanced 3D texts. You just need one spline of the tube profile and one spline which represents the path of the tube. See the image below.

Usage
To use the sweep object you just drag and drop two spline objects into the sweep object. The first spline (or child) will be used as the profile. The second spline (or child) will be used as the path. See also the images above. All children of the sweep object stay editable as always in Cheetah3D.

Properties

  • Maintain profile: If this property is turned on Cheetah3D tries to maintain the cross section of the swept profile in corners.
  • Flip normals: When off (by default), clockwise-winding splines result in the extruded polygons to be outward-facing. Tick this to flip the normals so that they face inwards instead.
  • Cover: Controls whether the endings of the hose will be closed or not.
  • Scale sweep: Scales the sweep towards the end of the path.
  • Rotate sweep: For maintaining horizontally orientation along the whole path.
