Cone Object

The cone is a parametric polygonobject which gives you a cone which you can modify in a number of different ways. It's basic form appears as below.

Properties

  • Sections Longitudinal: Specifies the number of sections the longitudinal distance (equator) of the cone will be made up of. Defaults to 30.
  • Sections Height: Specifies the number of sections the vertical axis of the cone will be made up of. Defaults to 1.
  • Sections Ground: Specifies the number of sections the base of the cone will be made up of. Defaults to 1.
  • Height: The height of the cone.
  • Radius: Specifies the radius of the cone at it's base.
