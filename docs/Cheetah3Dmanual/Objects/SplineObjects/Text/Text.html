Text Object

The text object is a parametric spline object. With help of the text object you can create a text spline out of any in Mac OS X installed True Type font. This text spline can than be used with help of the extrude or the sweep object to create a 3D Font.

Properties

  • Text: Specifies the text of the spline. To modify the font see above.
  • Font: Click on that button to get access to the Font panel.
  • Orientation: Specifies the axis to which the text spline will be perpendicular.
