テキストオブジェクト

テキストオブジェクトは、パラメトリックなスプラインオブジェクトです。このテキストオブジェクトを使用して、Mac OS X にインストールされたすべての TrueType フォントからテキストスプラインを作成する事ができます。（訳注：ヒラギノゴシック・ヒラギノ明朝などの OpenType フォントを使用する事も可能です。）テキストオブジェクトは、押し出し や パス押し出し と共に使用して、3D フォントの作成に使われる事が可能です。

プロパティ

  • テキスト: スプラインの文字列を入力します。フォントを指定するには、"Font" ボタンを使用します。
  • Font: このボタンをクリックし、フォントパネルからフォントを指定してください。
  • 方向: テキストが直交する軸を指定します。
