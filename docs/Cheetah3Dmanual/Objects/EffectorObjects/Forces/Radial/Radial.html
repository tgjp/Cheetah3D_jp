Radial Force

Radial Force effector objects moves particles in the direction of a circles radius in a 2D plane. Radial spinning particle storms can easily be created. Multiple effector types can be used to create even more complex behavior.


Radial Force effector object used with a particle emitter to create a circular particle effect.

Properties

  • Strength: Strength of the radial force. The radial force is around the Y axis of the objects local coordinate system.
  • Size: The size of the range.
  • Geometry: The shape of the range.
  • Invert: Inverts the range.
