Shear Modifier

The shear modifier moves the vertices of a mesh linearly in one direction proportionally to the height of each vertex. This modifier can be used to create cursive 3D Fonts for example, or to skew shapes so that they tilt.

Properties

  • Cage: See the description of the modifier object.
  • Type: See the description of the modifier object.
  • Distance: The maximum distance vertices are moved.
  • Angle: The direction in which the vertices will be moved.
