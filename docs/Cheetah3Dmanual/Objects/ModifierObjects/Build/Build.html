Build Modifier

The build modifier uses a certain percentage of the polygons from the original objects to create a new one. Using different types of build operations in conjunction with the wide range of parent objects possible, you can create some fascinating new objects using this modifier alone.


type=harlton, percentage=0.5

Properties

  • Type: Specifies the algorithm for choosing the polygons of the new mesh.
  • Percent: The percentage of polygons which will be used for creating the new object.
