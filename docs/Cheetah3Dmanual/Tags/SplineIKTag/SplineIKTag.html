スプライン IK タグ

スプライン IK タグは、ジョイントチェインをスプラインオブジェクトに形に沿わせたい場合に使用します。例えば、動物のしっぽを動かしたい場合、１つ１つ全てのジョイントではなく、スプラインのコントロールポイントを動かすだけでよくなります。

例:

例として、キャラクターの右手の IK チェインのセットアップ方法を見てみましょう。

  1. すでに骨組み（ジョイントチェイン）とスプラインを設定している事を確認して下さい。
  2. ジョイントチェインの先端を選択し、スプライン IK タグをツールバーから適用します。
  3. スプライン IK タグのターゲット プロパティにスプラインオブジェクトをセットします。
  4. これで完了です。スプラインの形状を変更すると、ジョイントチェインもそれに沿って動きます。

最初のイメージでは、スプラインとジョイントチェインが確認できます。

次のイメージでは、ジョイントチェインはスプライン IK タグによってスプラインに結びついています。
これでジョイントチェインは、曲線にスムースに沿っています。


プロパティ:

  • 長さ: IK チェインの長さ
  • オフセット: IK チェインのオフセット
  • ロール: IK チェインの回転角度。ジョイントをスプラインに沿ってひねらせたい場合にこの値を変更します。
  • ターゲット: ボーン（ジョイントチェイン）が沿うスプラインオブジェクト。
  • カラー: IK チェインのルートから IK ハンドルに接続するラインとスプライン IK ハンドルの色。
