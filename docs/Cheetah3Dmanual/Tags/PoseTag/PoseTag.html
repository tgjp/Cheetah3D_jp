ポーズタグ

ポーズタグは、よく使用するキャラクターのポーズを記録する場合に、とても便利です。例えば、あなたのキャラクターの右手用に"握りこぶし"を記録する事ができ、あなたがキャラクターに握りこぶしを作らせたいときには、ポーズ タグの "Set Pose" ボタンをクリックするだけで、キャラクターは握りこぶしを作るでしょう。

新しいポーズタグを作成するには、まず記録したいオブジェクトを選択する必要があります。例えば、もし手のポーズを記録したい場合には、手の手首のジョイントを選択します。次に選択された手首のジョイントにポーズタグを追加します。もしポーズタグの"新規ポーズ"ボタンをクリックすると、新しいボーズが記録されます。これによって手首ジョイントのすべての子オブジェクトの回転／スケールと位置が記録されます。

もしあなたがこのポーズを戻したい場合には、ポーズ マネージャの "Set Pose" ボタンをクリックします。

ポーズは主にキャラクターアニメーションで使われますが、シーン内のすべてのオブジェクトの移動変形（位置、回転、スケール）を記録するために使う事もできます。例えば、シーンを眺めるいくつかの決まったよいカメラ位置の記録に使用する事もできます。

プロパティ

  • ポーズ: 記録されたポーズの表
      • ポーズ名
      • もし "P" カラムのチェックボックスがチェックされている場合、"セット" ボタンがクリックされると記録されたポーズの位置がセットされます。
      • もし "R" カラムのチェックボックスがチェックされている場合、"セット" ボタンがクリックされると記録されたポーズの回転がセットされます。キャラクターのポーズを記録している場合には、位置やスケールの変更なしに、ジョイントの回転値を設定するだけで十分でしょう。"P" や "S" を無効にする事で、不必要なアニメーションキーが打たれる事を避ける事ができます。
      • もし "S" カラムのチェックボックスがチェックされている場合、"セット" ボタンがクリックされると記録されたポーズのスケールがセットされます。
      • ２番、３番、４番目のカラムのマスク情報を使用して、ポーズを設定します。
    • ポーズを更新: 現在のポーズで選択されたポーズを更新します。
    • 新規ポーズ: 新しいポーズを記録します。
