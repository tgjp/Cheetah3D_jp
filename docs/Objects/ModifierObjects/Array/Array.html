配列複製モディファイア

配列複製モディファイアはポリゴンオブジェクトにポリゴンのコピーを作成します。例えば超高層ビルをモデリングする場合に、これは大変便利です。おそらく、手作業で２０階分のフロアをモデリングしたくはないでしょう。配列複製モディファイアを使えば、１階フロアを作るだけでよくなります。後の１９階分は自動的に生成されます。ただ、このモディファイアは簡単にハイポリゴンオブジェクトを生成するので注意して下さい。


方向=X, コピー数=3, オフセット配置=1.0

プロパティ

  • 方向: コピーがそれに沿って作成される軸。
  • コピー数: 元のメッシュを含めた、コピーされる数。
  • オフセット配置:コピーされる間隔
