Boolean Modifier

The boolean object is a very powerful tool if you want to make some solid geometry. For example, you can subtract one object's area from another. There are currently three different boolean operations available. See the examples below for a clearer idea of how each boolean operation works.

Subtract

Usage
To use the boolean modifier, just arrange the polygon objects and the modifier like in the image below. The parent object of the modifier will be object A and the child object of the modifier will be object B. Now choose the boolean operation you want to perform from the properties editor (A add/subtract/intersect B).
