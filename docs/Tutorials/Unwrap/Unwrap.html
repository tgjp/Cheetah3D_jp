UV 展開

このチュートリアルでは複雑な形状のオブジェクトにUV座標値を設定します。そのためには、全ての3D空間の頂点に2次元のUV 座標値を設定しなければいけません。形状が複雑なほどUV 座標値を設定することは難しくなります。しかし、UV 展開ツールを使うことにより数回のマウスクリックでUV 座標値を設定できます。

1. 形状データの読み込み
まず最初に、形状データを読み込まなければいけません。Cheetah3Dフォルダーの"Example"フォルダーの中に"Pig"というファイル名で豚の形状データがあります。

まず、ウインドウレイアウトを"UV 編集"に切り替えます。これで、3D ビュー画面とUV エディタツールが表示されます。

オブジェクトブラウザで豚のオブジェクトのメッシュを選択するとUV エディタツールに豚のUV座標が表示されます。しかし、3番目の図のように形状が一部分重なっていたりいびつになっていたりしてUV座標値は不規則になっています。このようなUVは全て使い物になりません。そこで、形状が重なり合わない整然としたUVにする必要があります。

2. UV 展開ツールを使ってみる。
さて、UV 展開ツールを初めて使ってみよう。まず、 メニューから "ツール->UV ツール->UV 展開"を選択します。 すると、右図のような表示になります。しかし、表意jされたメッシュは一部は規則的に並んでいますがほとんどの箇所は形状が重なっています。あわてなくて大丈夫、これが正常なのです。

ゴムボールを想像してください。ゴムボールを切り裂かずに平らに広げることは出来ません。 出来ることは押しつぶして平らにすることだけです。 しかし、それでは上下の面が重なり合ったままです。決して平らに広がりません。そう、ボールを平らに広げるには 一部に切り目を入れる必要があります。そして、それは次の段階で正確に行う必要があります。

3. 切れ目の設定
UV 展開コマンドにより切れ目としてメッシュの端を指示します。 切れ目の指定の方法はハードエッジの指定より簡単です。（斧のチュートリアルを参照）切れ目の設定は切れ目にしたいエッジを選択し、メニューコマンドから "選択範囲->切れ目の設定/解除" を選択します。

今回の豚のケースではお腹の部分を切れ目とします。正常に指示したメッシュの端が切れ目と設定されたら線の色が緑色に変わります。

4. UV 展開ツールでの設定の調整
さて、UV 展開ツールで先ほどの切れ目の設定を使い調整します。再度メニューから "ツール->UV ツール->UV 展開"を選択します。

これで、ほぼ問題ない形状になりました。.体は全て大丈夫なようです。しかし、耳だけはメッシュが重なり合っています。

さて、ゴムボールの例えに戻ります。ボールを豚の形に置き換えてみましょう。もし、お腹の部分を切って押しつぶすと耳の部分は押しつぶしても持ち上がるでしょう。さて、どうしましょう？そう、耳の部分が広がるまで切れ目を引っぱって広げれば良いですね。 そして、ボールを木の板の上に置いて引っぱった部分を画鋲で固定できます。ただし、その場所は重なってはいけません。

これと同じ事をCheetah3Dの中で行うのです。

5. UV ピンの設定/解除と UV 展開の有効範囲座標
まず、UV ピンの設定/解除でUV 展開の有効範囲座標を固定しなければいけません。これは、とても簡単です。右図のように2カ所のポイントを選択ツールツールで選択します。次に、マウスの右ボタンをクリックしてコンテキストメニューから"UV ピンの設定/解除" を選択します。ポイントの選択を解除するとポイントの色が黄色に変わります。

6. UV 展開で最後の調整
ピンで固定した部分の移動する状態は右図の画面の中央に表示されています。注意して欲しいのは下の段の図を見れば分かるようにUV 展開コマンドを使用してもピンで固定された位置は変わらない事です。

それでは、最後に再度UV 展開ツールを選択します。これで、完璧な形状にUV展開した豚となりました。

これでテクスチャーの割り当て準備完了です。

ここからは、あなたが考えて決めることです。同じ要領で異なる形状のUV展開をあなたは出来るでしょう。例えば、耳の根元を切れ目とすれば、もっと良い形状のになったはずです。 きっと、均一の間隔のメッシュで構成されていたはずです。そう、ピンで固定しなくても大丈夫だったのです。
