"Material" シェーダー

"Material" シェーダーは、多くのマテリアルタイプの出発点です。テクスチャを貼付けたりするほか、ガラスや金属の作成にも使用します。

プロパティ:

カラー: これらのパラメータで、マテリアルの色を設定します。

  • color: マテリアルの色
  • texture: カラーテクスチャ
  • mix: カラーとテクスチャを混ぜ合わせるために使用します。

Ambient - Diffuse: これらのパラメータは環境光と拡散光の影響を調整するパラメータです。通常、反射や透過のないオブジェクトの場合、これらの値は 1.0 であるべきでしょう。しかし、ガラスや金属の場合（訳注：100 % 反射・透過する場合）には、これらの値は 0 であるべきです。

  • ambient: 環境光の影響を調整します。
  • diffuse: 拡散光の影響を調整します。

スペキュラー: これらのパラメータは鏡面反射の色とサイズを定義します。鏡面反射の強さは、"スペキュラー" か "texture" パラメータでコントロールする事ができます。白色は３つすべての各 RGB 要素について強さ 1 を意味し、黒は強さ 0 を意味します。

  • スペキュラー: 鏡面反射の色
  • texture: 鏡面反射のテクスチャ
  • mix: カラーとテクスチャを混ぜ合わせるために使用します
  • スペキュラーサイズ: 鏡面反射のサイズ。より大きいサイズは、鋭く小さい鏡面反射を作ります。

エミッシブ: 最終的なマテリアルのカラーに、発光要素を付け加えます。これはシーンの中で、ライトと相互には作用しません。

    • emission: 最終的なマテリアルのカラーに加えられる色。

バンプマップ: バンプマップは、表面の法線を置き換えます。これは、表面そのものを変更するものではありませんが、表面がライトの設定とどのように相互作用するかを変更します。したがって、実は平らな表面でも、ライトソースの下で凹凸があるように見えます。

    • bump map: 凹凸のテクスチャ（バンプマップ）を設定します。バンプマップは、白色が最も隆起した部分、黒色が隆起なしとして作成されたグレースケールのテクスチャです。そのためこれらのテクスチャは、高さテクスチャ（ハイトマップ）や浮き彫りテクスチャ（レリーフマップ）とも呼ばれます。
    • intensity: バンプ（隆起）の強さ（高さ）を設定します。
intensity=0.2
intensity=0.4
intensity=0.6
intensity=0.8
intensity=1.0

リフレクション: これらのパラメータで、どのくらい強くマテリアルが環境を反射（リフレクト）するかを設定する事ができます。例えば、鏡は 100 % 環境を反射するでしょう。環境から反射した色はマテリアルに加えられます。環境からの反射をフィルタリングするために "反射" や "texture" パラメータを使います。

    • 反射: 反射（リフレクション）のフィルターカラー
    • texture: 反射（リフレクション）のフィルターテクスチャ
    • mix: カラーとテクスチャを混ぜ合わせるために使用します
    • intensity: 反射の強さ。反射がマテリアルのカラーに加えられるので、色の飽和を避けるためにはカラーの "color" か "diffuse" プロパティを下げなければいけません。

    • angle: ぼやけた反射（リフレクションブラー）を作成するためにこのプロパティーを使用します。高い値ほどよりぼやけた反射になります。"反射のサンプル数" を上げる事は、ぼやけた反射のノイズを減らします。

    • 反射のサンプル数: ぼやけた反射の計算用のサンプル数。レンダリング時間は、このサンプル数とともに線形に増加します。

トランスパレンシー: トランスパレンシーパラメータは、ガラスのような透明なマテリアルを作成するために必要になるでしょう。"intensity" を上げると、色の飽和を避けるために自動的に "ambient" と "diffuse" の強さは下げられますが、それでも色ガラス（例えば赤いガラス）を作成する場合は、"ambient" と "diffuse" プロパティを 0 に設定する事が勧められます。

    • 透明度: 透過のフィルターカラー
    • texture: 透過のフィルターテクスチャ
    • mix: カラーとテクスチャを混ぜ合わせるために使用します
    • intensity: 透過の強さ
    • IOR: 屈折率が高いほど、光はマテリアルを通過する間により曲げられます。
    • アルファチャンネルを使用: カラーテクスチャのアルファチャンネル（や、カラーの透明度）を透過の計算に使用する場合にチェックします。
    • フレネル: 透過オブジェクトにフレネル反射を加える場合にチェックします。
