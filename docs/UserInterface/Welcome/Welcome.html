ようこそ！

はじめに、Cheetah3D をご利用いただきありがとうございます。うまくゆけば、あなたがこのアプリケーションを使用して学んでいくほど(あなたが 3D モデルに慣れていなくてもこの大量のドキュメントがあなたを導くでしょう)簡単で、シンプルで澱みなく、迅速で、使うことが楽しくなるほど効率的で、とてもパワフルであることを実感できるでしょう！

多くの 3D モデラやレンダラは非常に高価なだけではありませんが、同時にそれらは学ぶのが非常に難しいのが現状です。なぜなら、一部のソフトウエアメーカーたちはこのようなソフトウエアを使うのが難しいものであるとユーザに考えて欲しいらしく、とんでもない値段に耐えられる人たちだけのものとしたいのです。Cheetah3D が目指すのは、パワフルで簡単な 3D モデラとレンダラを皆様にご提供することです。

さらによいことに、Cheetah3D はゼロから MacOS X の Cocoa を使用して書かれています。それは Macintosh のソフトウェアらしくない振る舞いや操作性、見た目を持つ X11 のユーザーフレンドリーでないモデラーを、単純にリコンパイルしたものではありません。Cheetah3D は、家で心地よく使えるように、私たちがよく慣れ親しんだ Mac OS X コンポーネントを使用しています。

まず、何を？
 もし、あなたが 3D モデリングの初心者で基礎を学びたいのなら、ビギナーズガイドを読んでください。これは最初に学ばなければならない最も重要なものをあなたに教えるでしょう。そして次に、すぐに始められるように、あなたをいくつかの簡単なチュートリアルに導くでしょう。

もし、3D モデリングに慣れていてあなたの（別のソフトウェアで使っていた）方法を見つけたいなら、それを可能にするパワフルなツール群をインデックスから見つけてください。Cheetah3D があなたの望みに十二分に答えることを理解して頂けるでしょう。

もし、3D モデリングのベテランであり、あなたが Cheetah3D を利用して本当に印象的なものを作成するならば、レンダラの仕様を参照してください（使いやすいにも関わらず、 Cheetah3D が驚くほど素晴しいイメージを作り出すことが出来るのを理解して頂けるでしょう）。インターネット接続環境があれば、Cheetah3D で作られたイメージをオンラインギャラリーで見ることも出来ます。
