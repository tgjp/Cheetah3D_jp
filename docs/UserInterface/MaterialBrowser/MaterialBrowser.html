マテリアルブラウザ

マテリアルブラウザは、シーンで使われるすべてのマテリアルを管理します。マテリアルの追加、削除、また検索するために使用する事ができます。マテリアルブラウザのほとんどの機能は、コンテキストメニュー（ CRTL + クリック / 右クリック ）で使う事ができます。

マテリアルの追加 - "Add Material" ボタン
 プルダウンメニューボタンの "Add Material" を使う事によって、新しいマテリアルをシーンに追加する事ができます。 ボタンをクリックし、メニューから追加したいシェーダーを選択します。

マテリアルの削除 - "削除" コンテキストメニュー
マテリアルを削除するには、マテリアルブラウザでマテリアルを選択し、デリートキー（ delete ）を押します。

マテリアルの検索
 マテリアルブラウザは、マテリアルの名称やタイプでの検索機能を提供しています。例えば、右上の検索欄に "Wood" と入力すると、"Wood" シェーダーか名称に "wood" が含まれているもののみが表示されます。（訳注：この際、大文字小文字は無視されるようです。）

マテリアルを保存…
 後で再利用するために、あなたが一度作成した新しいマテリアルをマテリアルライブラリに保存したいかもしれません。そのために保存したいマテリアルを右クリックし、コンテキストメニューから "マテリアルを保存…" メニューコマンドを選択します。これでマテリアルライブラリにマテリアルが保存されました。
