細分化 キャトマルクラークツール

細分化 キャトマルクラークツールは編集中のポリゴンオブジェクトに対して単一のキャトマルクラークサブディビジョンプロセスを実行します。ポリゴンオブジェクトを滑らかにしたり、詳細を追加することができます。細分化 キャトマルクラークツールはすべてのハードエッジや選択状態 (マテリアル) のプロパティを保持します。

ハードエッジやポリゴン面の複数選択に関しての詳しい情報は、ロウ・ポリゴンオブジェクトを参照してください。

モード
 細分化 キャトマルクラークツールはロウ・ポリゴンオブジェクトにのみ使用可能です。ポイントモード、エッジモード、ポリゴンモードとオブジェクトモードの各モードで同じ動作をします。


  • To perform a subdivide, you just select a raw polygon object from the object browser and use the menu command "ToolsPolygonCatmull-Clark subdivide". Now a Catmull-Clark subdivision process will be performed on the entire mesh. Creases and multiple materials will be retained.
    If you are in polygon mode and you\'ve selected some polygons the subdivision will be only performed on the selected polygons.
