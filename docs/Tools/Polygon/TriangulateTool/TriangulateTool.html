三角ポリゴンで分割

下図のように、三角ポリゴンで分割は選択した全ての4角形ポリゴンまたはNゴンを三角ポリゴンに変換します。


モード
 三角ポリゴンで分割はポリゴンオブジェクトのみに使用できます。また、すべてのモードで使用できます。


  • まず、いくつかの ポリゴンオブジェクトを選択し三角ポリゴンで分割を適用します。ポリゴンを何も選択していない場合は全てのポリゴンを三角ポリゴンに変換します。三角ポリゴンで分割はメニューコマンドの "ツールポリゴン三角ポリゴンで分割"を選択することにより使用できます。選択している全ての４角ポリゴンとNゴンを三角ポリゴンに変換します。

キー

    • 無し

プロパティー

  • 無し
