法線を反転ツール

Cheetah3Dはポリゴンの曲面をもとに計算しているためポリゴンの法線の方向はとても重要です。他のアプリケーションで作成したモデルをCheetah3Dに取り込むと、時々ポリゴンの法線方向が逆転している場合があります。このような場合は、ポリゴンの向きが逆方向なので正常な面に対し暗い面として表示されます。法線を反転ツールは間違ったポリゴンの法線方向を逆転させることにより正常な状態に戻すことが出来ます。


Modes
 法線を反転ツールは編集中のポリゴンオブジェクトにのみ使用できます。また、ポリゴンモードでのみ使用可能です。


  • 法線を反転ツールを使うためには法線方向が間違っている ポリゴンオブジェクトを選択します。もし、ポリゴンオブジェクトを選択していない場合は全てのポリゴンの法線方向を反転します。 メニューコマンドの "ツールポリゴン法線を反転"を選択します。選択されたポリゴンで作られた曲面の法線方向が反転します。
