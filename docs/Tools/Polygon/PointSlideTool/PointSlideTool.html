ポイントをスライドツール

ポイントをスライドツールは隣接したエッジに沿ってポイントを動かす事が出来ます。


モード
 ポイントをスライドツールは編集中のポリゴンオブジェクトにのみ使用できます。 ポイント、エッジモード、ポリゴンモードで同じように動作します。


  • ポイントをスライドツールを使用するのはポリゴンオブジェクトのポイントをクリックするだけです。クリックしたポイントを共有するエッジが全てハイライトされます。ハイライトされたエッジに沿ってポイントをドラッグドロップできます。現在のポイントの移動位置はオフセット比率として表示されます。マウスボタンを押すことをやめると操作終了です。

キー

    • shift : シフトキーを押しながらマウスをドラッグするとエッジの端部の外側までポイントを移動できます。
