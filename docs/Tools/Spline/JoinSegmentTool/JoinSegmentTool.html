セグメントを繋ぐ ツール

１つのスプラインオブジェクトのパスを接続したい場合がしばしばあります。そのためにセグメントを繋ぐツールを使います。２つのベジエ曲線のセグメントを接続した例を下のイメージで見てください。

注意: 同じスプラインオブジェクト内のスプラインを接続する事ができます。２つの異なるオブジェクトのスプラインを接続したい場合、子オブジェクトの取り込みツールで片方のスプラインオブジェクトをもう一方のスプラインオブジェクトに読み込む必要があります。

編集モード
 セグメントを繋ぐ は全ての編集モードで使用できます。ポイントモードでしかコントロールポイントや選択範囲が見えないので、ポイントモードで使用するのが良いでしょう。このスプラインツールは、ロースプラインオブジェクトでのみ使用できます。


  • セグメントを繋ぐと使用するには、まず接続する２つのパスを選択する必要があります。最初のパスの任意のコントロールポイントを選択し、接続するもう１つのスプラインも同様に選択します。これで２つのパスは、それぞれコントロールポイントが選択された状態になりました。次に、メニューコマンド "ツールスプラインツールセグメントを繋ぐ" を呼び出して、適用します。これで、２つのパスは、１つのパスの終了点ともう１つのパスの開始点がつながるように接続されます。
