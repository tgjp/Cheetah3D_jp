ハードエッジ ツール

ハードエッジ は、角のたったスプラインの作成したい場合に、役に立つツールです。これはベジエ曲線タイプのスプラインの接線が長さ 0 になるように再配置します。下のイメージを見て下さい。

編集モード
 ハードエッジ は全ての編集モードで使用できます。ポイントモードでしかコントロールポイントや接線が見えないので、ポイントモードで使用するのが良いでしょう。このスプラインツールは、ロースプラインオブジェクトでのみ使用できます。


  • ハードエッジ を使用するには、まず選択ツールツールで、角ばらせたいスプラインのコントロールポイントを選択しなければなりません。選択した後に、"ツールスプラインツールハードエッジ" コマンドを呼び出します。選択した全ての接線は、角ばったスプラインが作られるように再配置されます。
