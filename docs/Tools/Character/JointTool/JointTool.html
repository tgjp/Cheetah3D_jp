ジョイントツール

ジョイントツールは簡単にスケルトンを作ることを助けます。従って、自分で全てのジョイントを配置する必要はありません。 ジョイントツールはあなたのためにそのように動作します。ジョイントツールはスケルトンの階層の的確な位置にジョイントを配置します。

モード
 このツールは全てのモード で操作します。


  • ジョイントツールを使用するには "ツール キャラクターツール ジョイントツール" をメニューから選択します。ジョイントツール は ツールのプロパティに表示されます。新たにジョイントを作成するには3D ビュー画面をクリックします。新たなジョイントはXY,YZ平面 またはXZ 平面に作成されます。そう、通常は正面図表示でジョイントツール を使用するのが最適です。
    現在アクティブなジョイントはオレンジ色で表示されます。新たなジョイントを3D ビュー画面をクリックし作成する場合は現在アクティブなジョイントの子供の階層に配置されます。そして、新たに作成されたジョイントがアクティブなジョイントになります。よって、武器や足を作ることも簡単です。
    アクティブなジョイントを変更する場合は既存のジョイントをクリックしてください。

キー

    • shift: shiftキーを押しながらマウスをクリックすると ルート ジョイントが作成できます。
    • esc: リセットツール。アクティブなジョイントをリセットします。

プロパティ

  • 描画モード: 描画モード は新たに作成したジョイントに使用できます。
  • カラー: カラー は新たに作成したジョイントに使用できます。
