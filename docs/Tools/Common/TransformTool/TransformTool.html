移動変形ツール

移動変形ツール は、すべての 3D モデラーの仕事馬であり、おそらくもっともよく使われるツールでしょう。このツールは３つのツールを１つに含んでいます。これを使って、ポイントやエッジ、ポリゴン、オブジェクトや、タグを中心に、移動、拡大縮小、回転する事が出来ます。

変形:
移動変形ツール は選択したものに対してのみ動作するので、それらを動かす前に、まずオブジェクトやポイントなどを選択する必要があります。それぞれ異なる移動変形タイプを切り替えるには、対応したガジェットをクリックする必要があります。移動ツールを使うには、円錐をクリックします。回転ツールを使うには、輪をクリックし、拡大縮小ツールを使うには、立方体をクリックします。それぞれ対応したシンボル（円錐、リング、立方体）がハイライトされるので、現在アクティブな変形タイプを確認する事ができます。実際のビジュアルは、下の画像で確認してください。

ガジェットモード

移動ガジェットと拡大縮小ガジェットは、選択した軸方向で制限して移動／拡大縮小に使われる事も出来ます。例えば、Y 軸の円錐をドラッグする場合、X 軸と Z 軸はロックされ、オブジェクトは Y 軸上でしか動きません。しかし、シフトキーを押しながら Y 軸の円錐をドラッグした場合、今度は Y 軸がロックされ、オブジェクトを X-Z 平面上で動かす事ができます。

制限無く自由に移動するには、変形ガジェットの中心をクリック＆ドラッグするか、SHIFT + ALT キーを押しながらドラッグします。

選択:
移動変形ツールツールは、オブジェクトとその構成要素を変形するだけでなく、それらを選択する事も可能です。マウスで移動変形ガジェット以外をクリックすると、移動変形ツールツールは、ペイント選択の様に動作し、ポリゴンやエッジなど、動作中の選択モードで選択する事ができます。

tweak modeモード:
ただ１つのポイントやポリゴンを調整したい場合には、tweak modeモードが便利でしょう。tweak modeモードは、オブジェクト全体以外のポリゴンやエッジのような構成要素のみを変形する事ができます。tweak modeモードを使用するには、まずすべてのポリゴン、エッジ、あるいはポイントの選択を解除します。次に、SHIFT + ALT キーを押したまま、マウスをメッシュ、又はスプラインの上に動かします。微調整可能な構成要素が、ハイライト表示されます。ハイライト表示された構成要素をクリック＆ドラッグすれば、最初のそれらの構成要素選択しておく必要なく、直接それらを操作する事ができます。メッシュのちょっとした微調整を、すばやく実行する事ができます。

ガジェットモード:
移動変形がジェットは、通常選択範囲の中央に配置されます。しかし、時々ガジェットが別の場所に配置された方が便利な場合があります。例えば、オブジェクトの底や、ポリゴンの端など。ガジェットを好みの位置に配置するために、ガジェットモードを使用する事ができます。ガジェットモードに入るには、移動変形ガジェットの中央の球をダブルクリックします。ガジェットモードに入ると中央の球の色はシアンに変化します。移動変形ガジェットを、選択範囲を変形させる事無く、任意に移動／回転する事ができます。自動ハイライト機能を使って、エッジやポイントに移動変形ガジェットをきっちりとスナップする事も出来ます。移動変形ガジェットを配置したら、再度中央の球をダブルクリックしてガジェットモードから抜けます。
ガジェットを通常の配置に戻すには、（移動変形ガジェットではなく）何も無い空間をクリックするか、ESC キーを押します。

スナッピングモード:
スナッピングが On の時、移動変形ガジェットの中央は、ガジェットをそれらに重なるように移動させる時、選択したスナッピング構成要素（ポイント、エッジ、等）にスナップされます。（Y 軸の円錐をドラッグした時のように）制限移動をした場合、オブジェクトはその制限した軸にのみスナップされます。この場合、ガジェットの中心ではなく、マウスポインターが、スナップポイントとして使用されます。

補助キー

    • Shift（シフト） + クリック: 選択範囲を解除せず、現在の選択範囲にポイント、エッジ、またはポリゴンを追加します。
    • Command（コマンド）+ クリック: 選択したポイント、エッジ、またはポリゴンを選択解除します。
    • Shift（シフト） + Alt（オルト） + ドラッグ: 自由移動、または均一拡大縮小
    • Command（コマンド） + ドラッグ: スプラインオブジェクトでのみ動作します。閉じたスプラインを開く時や、接線を分けるために使用します。

プロパティ

  • 座標系: このプロパティで、移動／拡大縮小／回転を実行する座標軸を、シーン全体のグローバル座標系か、オブジェクトのローカル座標系、（訳注：または、選択した要素の法線座標系）から選択する事が出来ます。
  • オートモード: ポリゴンオブジェクトを編集している場合、任意の編集モードをあらかじめ選択することなしに、ポイント、エッジ、ポリゴンを直接編集する事が出来ます。
  • スナッピング: スナッピングモードの切換え
    • なし: スナッピング Off
    • コンポーネント: それぞれ構成要素（ポイント、エッジ、ポリゴン）へのスナッピングを On にします。
    • ポリゴン表面: 表面へのスナッピングを On にします。この場合、移動変形ガジェットはポリゴンオブジェクトの表面にスナップします。
    • グリッド: グローバルなグリッドへのスナッピングを On にします。
  • ポイント: このプロパティが選択されている場合、ポリゴンオブジェクトやスプラインのポイントへのスナップが可能になります。
  • エッジ: このプロパティが選択されている場合、エッジの中央へのスナップが可能になります。
  • ポリゴン: このプロパティが選択されている場合、ポリゴンの中央へのスナップが可能になります。
  • オブジェクト位置: このプロパティが選択されている場合、オブジェクト位置へのスナップが可能になります。
  • グリッド幅: グリッドスナッピングが選択されている場合の、ラスタライズ単位を設定する事ができます。
